A Microsoft quer que 2026 seja o ano do Copilot no Windows 11. Após integrar ainda mais o assistente digital no Microsoft Edge, a empresa liberou a primeira atualização do Canal Preview do Windows 11 com a liberação de funções de PCs Copilot+ para todos os usuários.

A atualização do Windows 11 Preview é identificada pela build 26220.7535 (KB5072046) e traz como destaque a integração do Copilot com o Narrador para ler descrições de imagens e conteúdos da tela, o que é muito útil para pessoas com deficiência visual, mas somente estava disponível para PCs Copilot+ até agora.

Para iniciar a integração, basta pressionar a Tecla do Narrador + Ctrl + D obter uma descrição da imagem focalizada com o mouse ou teclado. Para receber uma descrição da tela toda, pressione a Tecla do Narrador + Ctrl + S.

![](https://t2.tudocdn.net/790491?w=660&h=422)

A Microsoft também está melhorando a função de Continuidade entre Dispositivos, que permite continuar uma atividade iniciada no seu celular usando um atalho na barra de tarefas do computador.

> Agora introduzimos um caminho de integração adicional que aproveita o Sistema de Notificação do Windows (WNS). Essa opção adicional expande o escopo dos usuários a quem você pode oferecer a Continuidade entre Dispositivos, ao mesmo tempo em que oferece a mesma experiência perfeita no Windows.

Por fim, agora administradores de TI podem desinstalar aplicativos Copilot de dispositivos gerenciados em PCs com Windows 11 Pro, Enterprise e Edu.
