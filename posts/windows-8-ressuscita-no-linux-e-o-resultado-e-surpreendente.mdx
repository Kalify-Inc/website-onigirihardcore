O Windows 8 pode não ter sido o sistema mais amado da Microsoft, mas ninguém pode negar que ele deixou sua marca, principalmente por causa dos usuários do Windows Phone. Agora, mais de uma década depois, esse visual icônico está ganhando uma nova vida no Linux, graças a um projeto open source chamado Win8DE.

O Win8DE é uma camada de interface completa inspirada no Windows 8, projetada para funcionar sobre gerenciadores de janelas Wayland, como Hyperland, Sway, Labwc e River. Isso significa que ele não é apenas uma launcher que imita o visual do sistema da Microsoft, mas recria elementos interativos da experiência original, como o menu Iniciar dinâmico, a tela de bloqueio e os controles de brilho e volume.

Apesar disso, o projeto não chega a ser um ambiente de desktop completo. O Win8DE funciona como uma shell leve, adicionando os principais recursos estéticos e funcionais do Windows 8 ao Linux. O destaque vai para o menu Iniciar, que oferece busca integrada, suporte a arrastar e soltar, e atalhos dinâmicos, tudo fiel à experiência original.

![](https://t2.tudocdn.net/791239)

Quem sempre gostou da ideia de uma interface fluida e moderna vai se sentir em casa. O Win8DE ainda inclui ferramentas de personalização, como troca de papel de parede e ajustes de bloqueio, mas deixa de fora a barra “Charms”, que marcou presença na versão original do Windows 8.

Mesmo assim, o projeto mostra como a comunidade Linux continua criativa e nostálgica, capaz de trazer de volta ideias que pareciam esquecidas. Para quem sente falta daquela mistura de inovação e ousadia que o Windows 8 tentou entregar, o Win8DE é uma nova chance de reviver essa experiência.

## Principais recursos do Win8DE

**Menu Iniciar**

- Comando Simples: Um comando (Win8Start) mostra ou esconde o menu e pode ser configurado na tecla "Super" (a tecla Windows do teclado).

- Arrastar e Soltar: Suporte total para organizar os blocos (tiles). Você pode mudar o tamanho deles (Pequeno, Médio, Grande, Extra Grande) clicando com o botão direito.

- Apps: Pode arrastar qualquer programa da lista "Todos os Apps" para os blocos da tela inicial.

- Busca: A barra de pesquisa de aplicativos funciona de verdade.

- Área de Trabalho: Se você arrastar um app para a parte de baixo, o menu fecha e o ícone vai para a área de trabalho (ou onde o sistema permitir).

- Menu de Energia: Clique no ícone do usuário para ver opções de desligar/reiniciar.

- Bateria: Mostra o nível de bateria na tela.

**Controles na Tela**

- Aquelas barrinhas que aparecem quando você muda o volume ou brilho.

- Funciona para: Volume (aumentar, diminuir, mudo) e Brilho (aumentar, diminuir).

- Nota técnica: Funciona em duas partes (servidor e cliente) que devem iniciar sozinhas.

**Papel de Parede**

- Suporte simples para imagens, configurável pelo menu de ajustes.

**Tela de Bloqueio**

- Estilo idêntico ao do Windows 8.

- Imagem de fundo personalizável.

- Tem aquela animação bonita de "deslizar" para cima e para baixo.

- Melhoria: Não precisa clicar e arrastar o mouse para desbloquear, apenas um clique já resolve (diferente do original que era chato).

**Configurações**

- Painel gráfico para mudar os papéis de parede das 3 áreas (Menu Iniciar, Área de Trabalho, Bloqueio).

- Dá para mudar as cores de destaque (accent colors) e cores de fundo para deixar com a sua cara.
